# Overview
This repository stems from the computer science capstone project of the Hyper Gesture Transfer Protocol team at the University of Utah.

The project depends on the following resources:
- A database
- An API
- A mobile app
- Some embedded device with gesture recognition and bluetooth capabilities

The project can not function as is without the resources above available.

There are also other optional resources created as part of the capstone which are available:
- A website
